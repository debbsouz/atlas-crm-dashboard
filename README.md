# Atlas CRM (Front-end)

Atlas CRM é um CRM de vendas desenvolvido em HTML, CSS e JavaScript, com foco em interface SaaS e experiência do usuário.

## Funcionalidades
- Login
- Dashboard com métricas
- Clientes: CRUD no front-end + busca + filtros
- Pipeline de vendas estilo Kanban (drag & drop)
- Histórico de atividades por cliente
- Relatórios com gráficos
- Importar/Exportar CSV
- Configurações (perfil e ações do sistema)
- Tema claro/escuro + sidebar institucional
- Modo Apresentação (para demo/prints sem apagar dados)

## Tecnologias
- HTML5, CSS3, JavaScript (Vanilla)
- LocalStorage para persistência
- Live Server para desenvolvimento

## Como rodar
1. Abra a pasta do projeto
2. Rode com Live Server (VS Code)
